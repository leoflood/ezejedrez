# Ezejedrez

A chess developed on React Native

---

## Requirements

For development, you will only need Node.js, npm, yarn and Expo (https://docs.expo.io/).

## Install

    $ npm install --global expo-cli
    $ git clone https://github.com/leoflood/ezejedrez
    $ cd ezejedrez
    $ yarn

## Test

    $ yarn test

## Start

    $ yarn start
